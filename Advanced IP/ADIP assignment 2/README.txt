The entire code is in assgn2.py

To run:

python assgn2.py <image file name> <mode>

mode = 0: image will be displayed, click left mouse button to get pixel location
Press esc to end.

mode = 1: image will be displayed, left mouse click for first point, then right mouse click to select second point. Distance between the two points will be displayed

mode = 2: select 4 points corresponding to two parallel lines. 
order: point 1 and point 2: end points of line 1
       point 3 and point 4: end points of line 2
output: vanishing point will be displayed

mode 3: projection matrix, extrinsic and intrinsic parameters will be displayed
